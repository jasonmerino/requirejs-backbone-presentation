Backbone.js Model Extras

Parsing data from forms/DOM can muck up data "types" and cause unnecessary model changes.

Our solution was to add a schema wrapper to Backbone.Model.

var AddressModel = Backbone.Model.extend({
  schema: {
    addressId: 'int',
    customerId: 'int',
    countryId: 'int',
    addressTypeId: 'int'
  }
});
